## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage                                                                             |
| ----- | ----- | --------------------------------------------------------------------------------- |
| xs    | 4px   | Icon gaps, inline padding, nav link gaps                                          |
| sm    | 8px   | Compact element spacing, chip gaps in skills grid                                 |
| md    | 16px  | Default element spacing, section internal padding, page horizontal padding (px-4) |
| lg    | 24px  | Section vertical padding, footer padding (py-6)                                   |
| xl    | 32px  | Terminal panel internal padding, remote-mount padding                             |
| 2xl   | 48px  | Space between major HomeView sections (hero to terminal, terminal to skills)      |
| 3xl   | 64px  | Page top/bottom padding (py-16)                                                   |

Exceptions: Terminal input prompt left-padding uses 12px (3 \* 4px) to align text after the `$` prompt character.

Source: Existing codebase patterns (px-4, py-16, py-6, p-8 all follow 8-point scale). Claude's discretion for exact spacing.

## Typography

| Role    | Size             | Weight            | Line Height          | Usage                                                                             |
| ------- | ---------------- | ----------------- | -------------------- | --------------------------------------------------------------------------------- |
| Body    | 16px (text-base) | 400 (font-normal) | 1.5 (leading-normal) | Bio paragraph, descriptions, terminal output                                      |
| Label   | 14px (text-sm)   | 400 (font-normal) | 1.5 (leading-normal) | Nav links, skill chips, footer social links, muted captions                       |
| Heading | 24px (text-2xl)  | 700 (font-bold)   | 1.2                  | Section headings (PlaygroundView title, skills section heading, terminal heading) |
| Display | 36px (text-4xl)  | 700 (font-bold)   | 1.1                  | Hero name only (Nick Tagliasacchi)                                                |

Terminal-specific typography:

| Element             | Font                    | Size           | Weight            |
| ------------------- | ----------------------- | -------------- | ----------------- |
| Terminal output     | monospace (`font-mono`) | 14px (text-sm) | 400 (font-normal) |
| Terminal input      | monospace (`font-mono`) | 14px (text-sm) | 400 (font-normal) |
| Terminal prompt `$` | monospace (`font-mono`) | 14px (text-sm) | 700 (font-bold)   |

Role trajectory ("Technical Lead -> Director of Engineering") uses 24px (text-2xl), weight 700 (font-bold), line-height 1.2. It shares the Heading size class but reads as secondary to the 36px display name due to context (subtitle position beneath hero name).

Source: Existing codebase (text-4xl, text-2xl, text-base, text-sm already in use). Claude's discretion for role trajectory size.

Glow effects:

- Terminal panel border: `box-shadow: 0 0 15px rgba(114, 241, 184, 0.15)` (cyan glow, subtle)

### TerminalPanel Interaction Contract

| Interaction          | Behavior                                                                                              |
| -------------------- | ----------------------------------------------------------------------------------------------------- |
| Mount                | Render pre-filled history from `cliDefaultOutput.json`. Scroll to bottom. Focus input.                |
| Type command + Enter | Append `{ input, output }` to history. Clear input. Scroll to bottom.                                 |
| `ls` command         | Print all keys from `cliCommands.json` with descriptions, formatted as `name -- description` per line |
| `help <cmd>`         | Print description for the named command, or `"Unknown command: <cmd>"` if not found                   |
| `whoami`             | Print `"Nick Tagliasacchi"`                                                                           |
| Unknown command      | Print error in destructive color: `"command not found: <name>. Type 'ls' to see available commands."` |
| Cursor               | Blinking block cursor in accent-yellow, CSS animation `blink 1s step-end infinite`                    |
| Scroll               | Terminal history area scrollable with `overflow-y: auto`, max-height `400px`                          |
| Mobile               | Full width, same max-height. Input remains fixed at bottom of terminal panel.                         |

### SocialLinks Interaction Contract

| Interaction         | Behavior                                                                                                      |
| ------------------- | ------------------------------------------------------------------------------------------------------------- |
| Render              | Read `socialLinks.json`, map `orientation` to flex justify class                                              |
| Hover on link       | Color transitions from `text-muted` to `text-accent` (pink)                                                   |
| Orientation mapping | `"center"` -> `justify-center`, `"left"` -> `justify-start`, `"right"` -> `justify-end`, `"none"` -> `hidden` |
| Icons               | Text labels only (no icon library). Format: link label as text with `<a>` tag wrapping.                       |
