Props:

```typescript
const props = defineProps<{
  skill: Skill
  mode: ProficiencyMode
  diamondSize: number // px (80 desktop, 56 tablet, 48 mobile)
}>()
```

Template structure:

- Outer div: `rotate-45 overflow-hidden rounded-sm border border-border transition-[transform,box-shadow,opacity] duration-200 ease-out` with `:style` binding for width/height from `diamondSize` prop
- Inner img: `-rotate-45 p-2 w-full h-full object-contain` with `:src="skill.iconPath"` and `:alt="skill.displayName"`
- The outer div binds dynamic classes for:
  - **Dimming (D-18):** When `!store.isSkillVisible(skill.category, skill.displayName)`, apply `opacity-30` class. When visible, apply `opacity-100`.
  - **Hover lift (D-06):** On mouseenter, apply `class` with `-translate-y-1 shadow-lg shadow-accent/30 z-10`. The translate and shadow create the lift effect. Note: translate MUST be combined with rotate-45 via style binding because Tailwind cannot stack transforms. Use `:style` for hover state: `{ transform: 'rotate(45deg) translateY(-4px)', boxShadow: '0 4px 16px var(--color-accent)' }` on hover, `{ transform: 'rotate(45deg)', boxShadow: 'none' }` on rest.
  - **Proficiency modes (D-12):** Computed `proficiencyStyle` based on `mode` prop:
    - `'uniform'`: empty object `{}`
    - `'glow'`: `{ boxShadow: '0 0 ${4 + skill.years * 2}px var(--color-accent)' }` (overrides hover shadow when not hovered)
    - `'size'`: `{ transform: 'rotate(45deg) scale(${0.85 + skill.years * 0.03})' }` (replaces base rotate transform)
    - `'fill'`: `{ backgroundColor: 'color-mix(in srgb, var(--color-surface-raised) ${Math.min(30 + skill.years * 7, 100)}%, transparent)' }`
  - Merge proficiency style with hover style. Hover takes precedence for boxShadow.

Events:

- `@mouseenter`: Set local `isHovered = true`, emit `'hover'` with `{ skill, element: $el }` for info panel positioning
- `@mouseleave`: Set local `isHovered = false`, emit `'leave'`
- `@click`: For mobile tap support (D-06 mobile behavior per UI-SPEC). Toggle `isHovered` state. If already hovered, set false. If not hovered, set true and emit `'hover'`.

Background color: `bg-surface-raised` (Tailwind utility using --color-surface-raised).

Accessibility: `aria-label` bound to `skill.displayName` on outer div.

**3. Create `apps/shell/src/components/skills/DiamondRow.vue`**

Per D-02, D-03, D-04, D-07, and UI-SPEC Continuous Scroll section.

Props:

```typescript
const props = defineProps<{
  skills: Skill[]
  speed: number // px per second (20, 25, 30, 35, 22 per UI-SPEC)
  rowIndex: number // 0-based, used for alternating offset
  diamondSize: number // px
  mode: ProficiencyMode
  isEntranceComplete: boolean // false until entrance animation finishes
}>()
```

Core mechanics:

- **Skill duplication for seamless loop:** Duplicate the `skills` array enough times to fill at least 2x viewport width. Computed `duplicatedSkills`:

  ```typescript
  const gap = 4 // px between diamonds
  const singleSetWidth = computed(
    () => props.skills.length * (props.diamondSize + gap),
  )
  const minWidth = typeof window !== 'undefined' ? window.innerWidth * 2 : 2560
  const repeatCount = computed(
    () => Math.ceil(minWidth / singleSetWidth.value) + 1,
  )
  const duplicatedSkills = computed(() => {
    const result: Skill[] = []
    for (let i = 0; i < repeatCount.value; i++) {
      result.push(...props.skills)
    }
    return result
  })
  ```

- **CSS animation duration:** `const duration = computed(() => singleSetWidth.value / props.speed)` -- this gives seconds for one full cycle.

- **Alternating row offset (D-01 masonry):** Odd-indexed rows (`rowIndex % 2 === 1`) get a negative top margin or translateX offset of half a diamond width to create masonry tight-packing. Use `:style="{ marginLeft: rowIndex % 2 === 1 ? '-${diamondSize / 2}px' : '0' }"` on the row wrapper.

- **Pause on hover (D-07):** Track `isPaused` ref. On `@mouseenter` of the row wrapper div, set `isPaused = true`. On `@mouseleave`, set `isPaused = false`. Apply class `paused` which sets `animation-play-state: paused` on the inner scrolling div.

- **Info panel state:** Track `hoveredSkill` ref and `panelPosition` ref. When a SkillDiamond emits `'hover'`, compute position via `getBoundingClientRect()` on the emitted element, set `hoveredSkill` and `panelPosition`. On `'leave'`, clear both.

- **Do NOT start scroll animation until entrance is complete.** Use `isEntranceComplete` prop. When false, the inner container has `animation: none`. When true, switch to `animation: scroll-row ${duration}s linear infinite`.

Template structure:

```vue
<div

  :class="{ paused: isPaused }"
  :style="{ height: `${diamondSize + 8}px`, marginLeft: rowIndex % 2 === 1 ? `${-(diamondSize / 2)}px` : '0' }"
  @mouseenter="isPaused = true"

    class="flex shrink-0 items-center gap-1"
```

```typescript
const diamondSize = ref(80)
const rowCount = ref(4)

function updateResponsive() {
  const w = window.innerWidth
  if (w < 640) {
    // mobile
    diamondSize.value = 48
    rowCount.value = 3
  } else if (w < 1024) {
    // tablet
    diamondSize.value = 56
```

```typescript
const rows = computed(() => {
  const allSkills: Skill[] = techSkills as Skill[]

  allSkills.forEach((skill, i) => {
    result[i % rowCount.value].push(skill)
  })
  return result
})
```
