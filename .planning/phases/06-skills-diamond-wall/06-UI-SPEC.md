## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage                                                 |
| ----- | ----- | ----------------------------------------------------- |
| xs    | 4px   | Gap between diamond grid items, icon internal padding |
| sm    | 8px   | Toolbar pill gap, search input internal padding       |
| md    | 16px  | Toolbar section spacing, info panel internal padding  |
| lg    | 24px  | Gap between toolbar and diamond wall                  |
| xl    | 32px  | Not used in this phase                                |
| 2xl   | 48px  | Not used in this phase                                |
| 3xl   | 64px  | Not used in this phase                                |

Exceptions:

- Diamond size: 80px (desktop), 56px (tablet), 48px (mobile) -- not on 8-point scale, but constrained to multiples of 8
- Diamond gap (between touching neighbors): 4px visual gap via CSS margin
- Hamburger menu touch target: 44px (existing pattern from TheHeader)

Usage in this phase:

- **Body (14px/400):** Search input text, info panel "years of experience" line, category text in info panel

| --------------- | ------------------------ | ------------- | ------------------------------------------------------- |

## Interaction Contracts

### Diamond Hover (D-05, D-06, D-07)

| State              | Visual                                                                                                                                   | Transition                             |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| Default            | Diamond at rest, scrolling with row                                                                                                      | --                                     |
| Hover enter        | Diamond lifts via `translateY(-4px)`, box-shadow increases, glow ring appears (accent color, 0 0 12px). Row scrolling pauses.            | 200ms ease-out                         |
| Info panel appears | Panel slides down from diamond bottom edge, 8px below diamond. Contains: skill name (16px bold), years (14px), category tag (12px cyan). | 150ms ease-out, 50ms delay after hover |
| Hover leave        | Diamond returns to rest position. Info panel fades out. Row scrolling resumes.                                                           | 200ms ease-in                          |

### Entrance Animation (D-08, SKL-03)

| Trigger                      | Behavior                                                                                                   |
| ---------------------------- | ---------------------------------------------------------------------------------------------------------- |
| Page load / scroll into view | IntersectionObserver on DiamondWall container. Once visible, diamonds animate in with staggered delays.    |
| Animation per diamond        | `opacity: 0 -> 1` + `scale(0.7) -> scale(1)` + `rotate(45deg)` (diamond rotation applied during entrance). |
| Stagger timing               | 30ms delay per diamond, row-by-row top-to-bottom, left-to-right within row.                                |
| Duration per diamond         | 400ms ease-out.                                                                                            |
| After entrance               | Continuous scroll animation begins per row.                                                                |

### Continuous Scroll (D-02, D-03, D-04)

| Property                  | Value                                                                                                                                   |
| ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| Direction                 | Left to right across full viewport width                                                                                                |
| Speed (row 1)             | 20px/second (slowest -- back row feel)                                                                                                  |
| Speed (row 2)             | 30px/second                                                                                                                             |
| Speed (row 3)             | 25px/second                                                                                                                             |
| Speed (row 4, if present) | 35px/second (fastest -- front row feel)                                                                                                 |
| Speed (row 5, if present) | 22px/second                                                                                                                             |
| Animation method          | CSS `@keyframes` with `translateX()`. Each row uses `animation: scroll-row linear infinite`.                                            |
| Looping                   | When a diamond fully exits right edge, it re-enters from left. If unique skills exhausted, loop existing. Duplicates acceptable (D-04). |
| Pause on hover            | `animation-play-state: paused` on the hovered row only (D-07).                                                                          |

### Filter/Search (D-15 to D-18, SKL-07)

| Interaction         | Behavior                                                                                                                                    |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| Category pill tap   | Toggle category on/off. Multiple categories can be active. "All" deselects individual categories; selecting any individual deselects "All". |
| Search input typing | Filter by displayName, case-insensitive substring match. Debounce 200ms.                                                                    |
| Filter applied      | Non-matching diamonds dim to `opacity: 0.3`. No layout reflow. Wall shape and scrolling continue uninterrupted (D-18).                      |
| Filter cleared      | All diamonds return to `opacity: 1` over 200ms transition.                                                                                  |
| Combined filter     | Category AND search must both match for a diamond to remain at full opacity.                                                                |

### Proficiency Mode (D-12, D-13, D-19, SKL-06)

| Mode              | Visual Effect                                                                                   | Formula                            |
| ----------------- | ----------------------------------------------------------------------------------------------- | ---------------------------------- |
| Uniform (default) | All diamonds identical. No proficiency indicator.                                               | --                                 |
| Glow              | Accent-colored glow ring. `box-shadow: 0 0 ${4 + years * 2}px var(--color-accent)`              | min 4px, max ~24px for 10+ years   |
| Size              | Diamond scales up. `scale(0.85 + years * 0.03)`                                                 | min 0.85x (0yr), max ~1.15x (10yr) |
| Fill              | Background opacity increases. `background: var(--color-surface-raised) / ${0.3 + years * 0.07}` | min 30% (0yr), max 100% (10yr)     |

Mode transitions: 300ms ease-in-out on all affected CSS properties.

## Responsive Contract

| Breakpoint                  | Diamond Size | Rows | Diamonds per Row    | Toolbar Layout                                                                 |
| --------------------------- | ------------ | ---- | ------------------- | ------------------------------------------------------------------------------ |
| Mobile (<640px / sm)        | 48px         | 3    | Fill viewport width | Stacked: search on top, pills horizontal-scroll below, mode toggle below pills |
| Tablet (640-1024px / md-lg) | 56px         | 4    | Fill viewport width | Single row: pills + search + toggle, pills scroll horizontally                 |
| Desktop (>1024px / xl)      | 80px         | 4-5  | Fill viewport width | Single row: all elements visible without scroll                                |

- Diamond wall always spans full viewport width (no max-w container) per D-02
- Toolbar constrained to `max-w-5xl mx-auto px-4` to match TheHeader layout pattern
- On mobile, hover interactions are replaced by tap: first tap shows info panel, second tap (or tap elsewhere) dismisses it
- Category pills use horizontal scroll with `-webkit-overflow-scrolling: touch` on mobile

## Accessibility Contract

| Requirement         | Implementation                                                                                                                                                             |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Keyboard navigation | Tab focuses toolbar controls. Arrow keys navigate category pills. Diamonds are decorative (not focusable individually).                                                    |
| Screen reader       | Diamond wall has `role="img"` with `aria-label="Technology skills showcase"`. Each diamond has `aria-label="{displayName}"`. Info panel uses `role="tooltip"`.             |
| Reduced motion      | `@media (prefers-reduced-motion: reduce)`: disable continuous scroll animation, disable entrance stagger, disable hover lift. Diamonds display statically in masonry grid. |
| Color contrast      | Text on `--color-surface-raised` background must meet WCAG AA (4.5:1). White (#fff) on #241b2f = 13.8:1 ratio. Muted text (#848bbd) on #241b2f = 5.2:1 ratio. Both pass.   |
| Touch targets       | Category pills: minimum 36px height with 8px padding. Search input: 40px height. Mode toggle buttons: 36px minimum.                                                        |

## Animation Performance Contract

| Constraint           | Requirement                                                                                                                                                        |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Animation properties | Only `transform` and `opacity` -- never animate `width`, `height`, `top`, `left`, or `margin`                                                                      |
| GPU compositing      | Diamonds use `will-change: transform` during scroll. Remove after entrance animation completes to free GPU memory.                                                 |
| CSS-only animation   | No JavaScript animation libraries (GSAP explicitly excluded per REQUIREMENTS.md Out of Scope). All animations via CSS `@keyframes`, `transition`, and `animation`. |
| Frame budget         | Scroll animation must maintain 60fps. Each row is a single composited layer.                                                                                       |
| Reduced motion       | All animations disabled via `prefers-reduced-motion: reduce` media query.                                                                                          |
