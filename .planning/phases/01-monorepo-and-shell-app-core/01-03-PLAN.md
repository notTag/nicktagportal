**3. Create `eslint.config.js`** (per D-01, RESEARCH Pattern 6):

This is the ONE allowed `.js` file in the entire project (SHELL-09 exception for ESLint config).

```javascript
import pluginVue from 'eslint-plugin-vue'
import {
  defineConfigWithVueTs,
  vueTsConfigs,
} from '@vue/eslint-config-typescript'

export default defineConfigWithVueTs(
  pluginVue.configs['flat/recommended'],
  vueTsConfigs.recommended,
  {
    files: ['**/*.vue', '**/*.ts'],
    rules: {
      // Allow unused vars prefixed with underscore
      '@typescript-eslint/no-unused-vars': ['error', { argsIgnorePattern: '^_' }],
    },
  },
  {
    ignores: ['**/dist/**', '**/node_modules/**'],
  }
)
```

Key details:
- Uses `defineConfigWithVueTs` helper from `@vue/eslint-config-typescript` (flat config format)
- Includes `pluginVue.configs['flat/recommended']` for Vue SFC rules
- Includes `vueTsConfigs.recommended` for TypeScript rules
- Ignores `dist/` and `node_modules/`
- File extension is `.js` (not `.mjs`) because root package.json has `"type": "module"` making all `.js` files ESM
