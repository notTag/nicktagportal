# Sketch Manifest

## Design Direction

Clean and minimal — Linear/Raycast vibe. Thin borders, generous whitespace, restrained motion, neutral palette with a single indigo accent. Sidebar lives as an **overlay** (content never shifts) and can be **dragged to dock** on the left or right edge.

## Reference Points

- Linear — subtle animation, neutral surfaces, tight typography
- Raycast — overlay panels, minimal chrome, deliberate motion
